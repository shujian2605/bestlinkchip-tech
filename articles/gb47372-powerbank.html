前言

GB 47372-2026《便携式数字设备用移动电源安全技术规范》已于2026年发布，2027年4月1日起强制执行。

这意味着：所有在中国市场销售的移动电源、充电宝，必须在2027年4月1日前通过新国标认证。不符合标准的产品将被禁止销售。

对方案商和品牌方来说，这件事不只是"多做一个认证"——它决定了你现在选的PCBA方案未来一年能不能卖。

本文从PCBA方案商的角度，拆解新国标的核心要求、技术应对方案、以及三条不同价位的合规路径。

一、GB 47372-2026 五大核心要求

我先把标准文件中的关键技术条款翻译成"做方案的人能听懂的话"：

条款 原文要求 对PCBA方案意味着什么
电芯针刺试验 电芯被刺穿后不起火、不爆炸 必须选用通过针刺测试的电芯，成本+10-20%
循环老化测试 500次充放电循环后容量≥80% 充放电管理芯片要有精准的电量计和老化补偿算法
唯一编码 每台移动电源必须有唯一的可追溯编码 主控芯片需支持写入和读取唯一ID（Flash/EEPROM）
智能热管理 充电温度不超过60℃，放电不超过70℃ 必须有NTC温度检测+BMS过温保护，不能省这个钱
禁止梯次利用电芯 不能用汽车拆下来的旧电芯 这条主要管供应链，不是技术问题，但要留好电芯来源凭证

二、新国标对PCBA方案的三个硬性要求

2.1 电源管理与保护——按功率段配套，杰理负责信息交互层

新国标前的很多低价充电宝，用的是基础充电IC（如ETA1038），只做充电，不做电池管理。

新国标要求的过充保护、过放保护、过温保护、短路保护，必须在硬件层面完整实现。佳联芯对外推荐的核心不是某个第三方电源芯片，而是杰理主控负责显示、蓝牙APP、唯一编码、产线写入和状态交互，电源管理/保护器件按功率段、认证要求和客户供应链配套。

  • 杰理芯片负责智能显示、BLE APP、唯一编码、异常状态读取和OTA等交互层能力
  • 充放电、电芯保护、NTC温控等电源安全部分按项目需求配套
  • 对外沟通只推荐杰理平台，不推荐其他芯片品牌

2.2 唯一编码——要有地方存

每台移动电源的PCB上需要存储一个唯一ID。这个ID可以通过蓝牙APP读取（给消费者看），也可以通过产线工具写入（给工厂用）。

  • 利用主控芯片的Flash区域存储唯一编码
  • 通过BLE（低功耗蓝牙）让手机APP读取
  • 产线烧录工具写入，每台一个号

2.3 温度监控——NTC必须留

很多老方案为了省钱，板子上没有NTC温敏电阻。新国标下这是红线。

  • 电池端 + PCB端各放一颗NTC
  • BMS芯片实时读取温度数据
  • 超过阈值自动降功率或切断输出

五、选型建议：按你的客户类型选

如果你是跨境电商卖家

  • 欧美日市场要求高 → 路线三（彩屏+APP，用户感知强）
  • 东南亚/拉美走量 → 路线一（成本优势，量大）

如果你是国内品牌方

  • 电商平台（天猫/京东/拼多多） → 路线二或三（需要差异化）
  • 礼品渠道/企业定制 → 路线三（彩屏加分）
  • 共享充电宝运营商 → 路线二（需要APP+BMS数据）

如果你是硬件创业团队

  • 第一款产品 → 路线二（稳妥+有卖点）
  • 预算紧 → 路线一（先活下来）

六、一个时间提醒

从现在（2026年6月）到强制执行的倒推时间线：

``
2026年6月 → 方案选型、PCB设计
2026年7-8月 → 打样、测试、送检
2026年9-10月 → 认证完成、小批量试产
2026年11-12月 → 量产备货（赶双11/黑五）
2027年1-3月 → 渠道铺货
2027年4月1日 → 强制执行，不合规产品下架
``
